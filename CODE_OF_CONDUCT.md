# Community Code of Conduct

We want this project to be a productive, respectful place for anyone who wants to improve SQL Server workflows with MCP. Be kind, assume good intent, and keep discussions focused on the work. Harassment, discrimination, or abusive behavior is not tolerated.

If you experience or observe behavior that violates these expectations, please email [connor.r.england@gmail.com](mailto:connor.r.england@gmail.com). Issues can also be reported privately via GitHub Direct Messages.

By participating in this project you agree to follow this policy.
