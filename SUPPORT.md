# Support

Need help with the MSSQL MCP server? Start with the options below.

## Issues & feature requests

- Search existing GitHub issues to see if your problem has already been reported.
- Open a new issue with repro steps, logs, and environment details (Node/.NET version, SQL target, etc.).

## Direct contact

For sensitive or private questions email [connor.r.england@gmail.com](mailto:connor.r.england@gmail.com). Please include links to any relevant issues or pull requests.

Community-based support is provided on a best-effort basis. There is no paid SLA.
