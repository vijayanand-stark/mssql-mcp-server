# Contributing

Thanks for helping improve the MSSQL MCP server! Contributions of any size are welcome—bug reports, docs, or new features.

## How to contribute

1. Open an issue describing the problem or enhancement so we can discuss scope and impact.
2. Fork the repo and create a feature branch.
3. Run tests/linters relevant to your change.
4. Open a pull request referencing the issue. Describe your change and include screenshots/logs if helpful.

By submitting a PR you confirm that you have the right to contribute the code and that it will be released under the project’s MIT license.
